About SHOPPEE

A clean, simple single-page shopping experience.

What is this project?

This project was created for COMP 3512 Web II as a single-page application that fetches product data and dynamically updates the page based on user interaction. This project uses TailWind CSS for styling.

Technologies Used

  • HTML
  • CSS (TailWind)
  • JavaScript - SPA logic (fetch, DOM, LocalStorage, etc.)
